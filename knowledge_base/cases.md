# Кейсы и проекты DevPunks

## AI-ассистент для бизнеса

Разработали чат-ассистента для компании, который отвечает на вопросы клиентов 24/7, используя базу знаний компании. Интегрировали голосовой интерфейс через ElevenLabs для более живого общения.

**Результат:** Снижение нагрузки на поддержку на 60%, NPS клиентов вырос.

## Автоматизация обработки документов

Построили систему автоматической обработки входящих документов: извлечение данных, классификация, маршрутизация — всё через LLM pipeline.

**Результат:** Сократили время обработки с 2 часов до 3 минут.

## Telegram-бот для мониторинга

Создали систему мониторинга и уведомлений с AI-анализом аномалий, интегрированную с Telegram.

## Семантический поиск для e-commerce

Внедрили векторный поиск в интернет-магазин — покупатели находят товары по описанию, не зная точных ключевых слов.

**Результат:** Конверсия из поиска выросла на 35%.

## Быстрые прототипы

DevPunks специализируется на быстрых MVP — за 2-3 недели создаём работающий прототип с реальным AI, который можно показать пользователям и инвесторам.
