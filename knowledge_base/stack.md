# Технический стек DevPunks

## AI / LLM

- **Anthropic Claude** (claude-sonnet-4-6, claude-opus-4-6) — основная модель для сложных задач
- **OpenAI GPT-4o / GPT-4o-mini** — альтернативы, выбор под задачу
- **LangChain / LangGraph** — оркестрация агентов и цепочек
- **RAG** (Retrieval-Augmented Generation) — знания из документов
- **Qdrant, Pinecone** — векторные базы данных

## Backend

- **Python / FastAPI** — основной стек для AI-сервисов
- **Node.js / Express** — для высоконагруженных API
- **PostgreSQL, MongoDB** — реляционные и документные БД
- **Redis** — кэширование и очереди
- **Docker, Docker Compose** — контейнеризация

## Frontend

- **Next.js / React** — основной frontend стек
- **TypeScript** — всегда, без исключений
- **Tailwind CSS** — стилизация
- **Vanilla JS** — для лёгких MVP и лендингов

## Voice AI

- **Vapi.ai** — WebRTC + STT для голосовых агентов
- **ElevenLabs** — эмоциональный TTS (text-to-speech)
- **Deepgram** — альтернативный STT

## DevOps / Инфраструктура

- **VPS (Hetzner, DigitalOcean)** — хостинг
- **Nginx** — reverse proxy, SSL
- **GitHub Actions** — CI/CD
- **Let's Encrypt** — SSL сертификаты

## Интеграции

- Telegram Bot API, WhatsApp API
- Stripe / платёжные системы
- CRM системы (Airtable, Notion)
- Webhooks, REST API, GraphQL
