# DevPunks — AI-first company website

Minimalist homepage with an AI agent chat interface. Inspired by ChatGPT's UX.

## Stack

- Vanilla HTML/CSS/JS (MVP preview)
- Next.js + TypeScript + Tailwind (coming in production)
- Claude API for the AI agent

## Features

- AI chat interface with typing indicator
- Dark / Light theme toggle
- EN / RU language switch
- DevPunks logo with brand colors (`#E50051` accent)
